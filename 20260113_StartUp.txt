Step 0 — 建立虛擬環境與安裝 Django
# 建立專案資料夾
mkdir ReceiptAI_Project
cd ReceiptAI_Project

# 建立 Python 虛擬環境
python -m venv venv
# Windows
venv\Scripts\activate
# macOS / Linux
# source venv/bin/activate

# 安裝 Django 與 Django REST Framework
pip install django djangorestframework pillow

pillow 用來處理圖片，OCR / QR 會用到。

===== ===== ===== ===== =====
Step 1 — 建立 Django 專案

django-admin startproject ReceiptAI_Project .

- 這會建立：
ReceiptAI_Project/
├── manage.py
├── ReceiptAI_Project/
│   ├── __init__.py
│   ├── settings.py
│   ├── urls.py
│   └── wsgi.py

===== ===== ===== ===== =====
Step 2 — 建立 apps
依照模組化設計建立 apps：
python manage.py startapp api
python manage.py startapp services
python manage.py startapp domain
python manage.py startapp infrastructure

- 結構大概長這樣：
ReceiptAI_Project/
├── manage.py
├── ReceiptAI_Project/
│   ├── settings.py
│   └── urls.py
├── api/
│   ├── views.py
│   ├── urls.py
│   └── serializers.py
├── services/
│   ├── qr_service.py
│   ├── ocr_service.py
│   ├── invoice_flow.py
│   └── classify_service.py
├── domain/
│   ├── invoice.py
│   ├── item.py
│   └── enums.py
├── infrastructure/
│   ├── google_sheets.py
│   └── storage.py

- 每個 app 都要在 ReceiptAI_Project/settings.py 的 INSTALLED_APPS 加入
INSTALLED_APPS = [
    ...
    'rest_framework',
    'api',
    'services',
    'domain',
    'infrastructure',
]

===== ===== ===== ===== =====
Step 3 — 設定基本 URL

- 在 ReceiptAI_Project/urls.py

from django.contrib import admin
from django.urls import path, include

urlpatterns = [
    path('admin/', admin.site.urls),
    path('api/', include('api.urls')),
]

- 在 api/urls.py

from django.urls import path
from . import views

urlpatterns = [
    path('invoice/scan/', views.scan_invoice, name='scan_invoice'),
]

===== ===== ===== ===== =====
Step 4 — 建立 API View Skeleton

- 在 api/views.py

from rest_framework.decorators import api_view
from rest_framework.response import Response

@api_view(['POST'])
def scan_invoice(request):
    """
    接收前端傳來的 Raw JSON 或圖片
    目前先回傳測試訊息
    """
    data = request.data
    return Response({"status": "success", "received_data": data})

- 測試 Django 專案是否可以運行

python manage.py runserver
